# 02_Knowledge_Base

Центральная база знаний о компании, клиентах, продуктах и процессах.

**Подразделы:**
- `Audiences/` — портреты целевых аудиторий
- `Clients/` — клиентская база, кейсы, отзывы
- `Processes/` — бизнес-процессы и регламенты
- `Products/` — каталог оборудования и описания
- `Team/` — структура команды, роли
- `Vendors/` — поставщики и условия работы
