# 08_AI_Context

Структурированные .md-файлы с выжимками знаний о компании — для работы с Claude и другими AI-инструментами.

**Что сюда входит:**
- company_profile.md — профиль компании
- brand_guidelines.md — бренд и tone of voice
- products.md — каталог и описания
- target_audience.md — аудитории
- sales_context.md — контекст продаж
- competitors.md — конкуренты
